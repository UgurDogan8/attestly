# Marketplace listing — Security section (short summary)

*Atlassian Marketplace listings have a dedicated "Security" free-text section, separate from
the linked privacy policy. Paste a version of the text below there (trim/adjust to fit the

---

Attestly is built on Atlassian Forge and runs entirely inside Atlassian's cloud
infrastructure — we operate no servers of our own and store no customer data outside of
Forge's built-in storage. All data stays within your site's Atlassian data residency region.

We request the minimum permissions needed to function: reading page metadata (title,
version), storing acknowledgement/assignment records, and posting comments to send in-app
reminder notifications via Confluence's own @mention system — we never use an external email
or SMS provider, so no customer data leaves the Atlassian platform.

The one feature that produces a downloadable file (the CSV audit export) is protected by a
secret token in addition to Forge's own unguessable web trigger URLs, so exports can only be
requested from inside the app's admin panel.

Full privacy policy: https://ugurdogan8.github.io/attestly-legal/
